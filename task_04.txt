В аттаче находится файл с Байесовской сетью. Формат следующий:
В первой строке перечислены вершины графа, нумерация происходит с нуля.
Далее идут строки с априорными вероятностями для вершин без предков.
Затем идет информация о каждой из вершин по порядку в следующем виде:

номер вершины : вершины, в которые из неё ведут ребра (пусто, если потомки отсутствуют).
список вероятностей в зависимости от конфигурации предков (пусто, если предки отсутствуют).

Требуется реализовать проверку сети на корректность, визуализацию графа, а также уметь делать вывод апостериорных вероятностей по введенным вручную свидетельствам.
